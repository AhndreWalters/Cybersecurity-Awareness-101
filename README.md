## Overview

Cybersecurity Awareness 101 is an educational website developed at T.A. Marryshow Community College to raise awareness about cybersecurity threats, phishing, and online safety practices.

## Project Structure

- **Home Page** - Introduction to cybersecurity importance
- **Common Threats** - Information about various cybersecurity threats
- **Prevention Tips** - Best practices and safety recommendations
- **Responsive Design** - Optimized for all devices

## Technology

- **HTML** - Website structure and content
- **CSS** - Styling and visual presentation
- **GitHub Pages** - Hosting and deployment

## Deployment

Automatically deployed via GitHub Pages using the included workflow configuration.

## Educational Purpose

This project serves as a practical resource for understanding:
- Common cyber threats and attack vectors
- Phishing recognition techniques
- Essential online safety measures
- Preventive cybersecurity practices
